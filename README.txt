(2) Getting Started:

These instructions will help you run on your local machine for development and testing purposes. 

(3) Prerequisites:

You need to install the software..

# https://www.jetbrains.com/pycharm/download/#section=windows

After setting up the software open the pycharm terminal nad run these commond to download pulgins

Install Django..

# pip install Django

Perform database migration..

# python manage.py check
# python manage.py migrate

Run Development Server..

# python manage.py runserver

install Anaconda computer program..

# pip install anaconda

There are more plugins you can download from setting as your requirement.
